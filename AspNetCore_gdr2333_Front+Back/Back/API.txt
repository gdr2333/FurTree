验证码操作：
    获取验证码：GET /Capcha/GetCapcha
        发送：
            Query：无
            Body：无
            Header：无
        接收：
            正常处理：
                返回码：200 OK
                内容：
                    Image：
                        类型：string(base64)
                        内容：base64编码的验证码图片（格式为image/png）
                    Id：
                        类型：string(guid)
                        内容：验证码Id

账号操作：
    创建账号：PUT /Account/Create
        发送：
            Query：无
            Body：
                Name：
                    类型：string
                    内容：用户名
                Email：
                    类型：string
                    内容：用户邮箱
                PasswordHash：
                    类型：string(base64)
                    内容：用户密码哈希值，算法为base64(sha256(utf8encode(password)))
                CapchaId：
                    类型：string(guid)
                    内容：验证码Id
                CapchaResult：
                    类型：string
                    内容：用户对验证码的输入
            Header：无
        接收：
            正常处理：
                返回码：204 No Content
                内容：无
            用户名已存在/用户邮箱已存在：
                返回码：409 Conflict
                内容：无
            验证码无效（过期/不存在）/验证码错误：
                返回码：400 Bad Request
                内容：无
    请求重发确认邮件：PUT /Account/ResendConfirm
        发送：
            Query：无
            Body：
                Email：
                    类型：string
                    内容：用户邮箱
                CapchaId：
                    类型：string(guid)
                    内容：验证码Id
                CapchaResult：
                    类型：string
                    内容：用户对验证码的输入
            Header：无
        接收：
            正常处理：
                返回码：204 No Content
                内容：无
            验证码错误：
                返回码：400 Bad Request
                内容：无
            邮箱未注册：
                返回码：404 Not Found
                内容：无
            邮箱已确认：
                返回码：409 Conflict
                内容：无
            重发请求过于频繁：
                返回码：429 Too Many Requests
                内容：无
            账号被锁定：
                返回码：403 Forbidden
                内容：无 
    登录账号：POST /Account/Login
        发送：
            Query：无
            Body：
                Name：
                    类型：string
                    内容：用户名或邮箱
                PasswordHash：
                    类型：string(base64)
                    内容：用户密码哈希值，算法为base64(sha256(utf8encode(password)))
                CapchaId：
                    类型：string(guid)
                    内容：验证码Id
                CapchaResult：
                    类型：string
                    内容：用户对验证码的输入
            Header：无
        接收：
            正常处理：
                返回码：200 OK
                内容：
                    类型：string
                    内容：JWT令牌，有效期7天
            验证码无效（过期/不存在）/验证码错误：
                返回码：400 Bad Request
                内容：无
            密码错误：
                返回码：401 Unauthorized
                内容：无
            账号被锁定：
                返回码：403 Forbidden
                内容：无
            用户不存在：
                返回码：404 Not Found
                内容：无
    更改账号名：PUT /Account/Name
        发送：
            Query：无
            Body：
                类型：string
                内容：用户名
            Header：
                Authorization：该用户登录获取的JWT
        接收：
            正常处理：
                返回码：204 No Content
                内容：无
            没有提供JWT/JWT无效：
                返回码：401 Unauthorized
                内容：无
            用户名冲突：
                返回码：409 Conflict
                内容：无
            新名称无效：
                返回码：400 Bad Request
                内容：无
    更改账号邮箱：PUT /Account/Email
        发送：
            Query：无
            Body：
                类型：string
                内容：新邮箱
            Header：
                Authorization：该用户登录获取的JWT
        接收：
            正常处理：
                返回码：204 No Content
                内容：无
            没有提供JWT/JWT无效：
                返回码：401 Unauthorized
                内容：无
            邮箱冲突：
                返回码：409 Conflict
                内容：无
            新邮箱无效：
                返回码：400 Bad Request
                内容：无
        备注：会重置邮箱确认状态并向新邮箱发送确认请求，如果新邮箱不是你的或者无效你就死定了，找管理吧
    更改账号密码：PUT /Account/Password
        发送：
            Query：无
            Body：
                类型：string(base64)
                内容：新密码哈希值，算法为base64(sha256(utf8encode(password)))
            Header：
                Authorization：该用户登录获取的JWT
        接收：
            正常处理：
                返回码：204 No Content
                内容：无
            没有提供JWT/JWT无效：
                返回码：401 Unauthorized
                内容：无
            密码相同：
                返回码：409 Conflict
                内容：无
